## 📚 调酒技巧

<div class="tips-section">

### 🎯 基础技巧

<div class="tip-box">
  <h4>1. 摇晃 (Shake)</h4>
  <p><strong>适用：</strong>含有果汁、糖浆、蛋清的鸡尾酒</p>
  <p><strong>时间：</strong>10-15秒，听到冰块碰撞声音变小即可</p>
  <p><strong>效果：</strong>充分混合，产生气泡，降温快</p>
</div>

<div class="tip-box">
  <h4>2. 搅拌 (Stir)</h4>
  <p><strong>适用：</strong>纯烈酒调配的鸡尾酒（如马天尼）</p>
  <p><strong>时间：</strong>15-20秒，保持顺滑口感</p>
  <p><strong>效果：</strong>温和混合，保持酒液清澈</p>
</div>

<div class="tip-box">
  <h4>3. 分层 (Layer)</h4>
  <p><strong>技巧：</strong>使用吧勺背面，沿杯壁缓慢倒入</p>
  <p><strong>原理：</strong>利用不同液体的密度差</p>
  <p><strong>顺序：</strong>从密度大到密度小依次倒入</p>
</div>

</div>

### 💡 进阶技巧

<div class="advanced-tips">

**冰块的选择：**
- 🧊 **大冰块**：融化慢，适合慢饮的威士忌类
- 🧊 **碎冰**：降温快，适合Mojito等清爽型
- 🧊 **冰球**：表面积小，稀释最慢

**柑橘的处理：**
- 🍋 榨汁前先滚压，可以榨出更多汁液
- 🍊 橙皮装饰时要挤压，释放精油
- 🍋 柠檬皮可以在杯口擦拭，增加香气

**薄荷的使用：**
- 🌿 轻拍释放香味，不要过度捣碎
- 🌿 使用前用手拍打，激发香气
- 🌿 装饰时选择完整美观的叶片

</div>

function calculateABV() {
  const alcoholPercent = parseFloat(document.getElementById('alcohol-percent').value);
  const alcoholVolume = parseFloat(document.getElementById('alcohol-volume').value);
  const totalVolume = parseFloat(document.getElementById('total-volume').value);
  
  if (isNaN(alcoholPercent) || isNaN(alcoholVolume) || isNaN(totalVolume)) {
    document.getElementById('result').innerHTML = '<p class="error">请输入有效的数值</p>';
    return;
  }
  
  if (totalVolume < alcoholVolume) {
    document.getElementById('result').innerHTML = '<p class="error">总液体量不能小于基酒用量</p>';
    return;
  }
  
  const abv = (alcoholPercent * alcoholVolume) / totalVolume;
  const pureAlcohol = (alcoholVolume * alcoholPercent) / 100;
  
  let strength = '';
  let strengthColor = '';
  if (abv < 10) {
    strength = '低度';
    strengthColor = '#28a745';
  } else if (abv < 20) {
    strength = '中度';
    strengthColor = '#ffc107';
  } else if (abv < 30) {
    strength = '高度';
    strengthColor = '#fd7e14';
  } else {
    strength = '烈性';
    strengthColor = '#dc3545';
  }
  
  document.getElementById('result').innerHTML = `
    <div class="result-display">
      <h4>计算结果</h4>
      <div class="result-item">
        <span class="result-label">酒精度：</span>
        <span class="result-value" style="color: ${strengthColor}; font-weight: bold;">${abv.toFixed(1)}%</span>
      </div>
      <div class="result-item">
        <span class="result-label">强度：</span>
        <span class="result-value" style="color: ${strengthColor};">${strength}</span>
      </div>
      <div class="result-item">
        <span class="result-label">纯酒精量：</span>
        <span class="result-value">${pureAlcohol.toFixed(1)}ml</span>
      </div>
    </div>
  `;
}
</script>
