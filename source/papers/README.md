# 论文PDF存放说明

将你的论文PDF文件放在这个文件夹中。

## 文件命名建议

- `paper1.pdf` - 第一篇论文
- `paper2.pdf` - 第二篇论文
- `thesis.pdf` - 学位论文
- `compression-algorithm-2024.pdf` - 具体论文名称

## 在页面中引用

在 Markdown 文件中这样引用：

```markdown
[下载PDF](/papers/paper1.pdf)
```

或者：

```markdown
<a href="/papers/paper1.pdf" download>📄 下载论文</a>
```
